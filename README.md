# E-Commerce Website Manual Testing Project

This repository contains manual test cases created for an E-commerce web application.

Modules Covered:
- Login Page
- Registration Page
- Homepage
- Cart
- Checkout
- Payment

Test Case Format Includes:
- Test Case ID
- Test Scenario
- Test Description
- Pre-condition
- Test Steps
- Test Data
- Expected Result
- Status
- Priority

Tools Used:
- Excel
- Manual Testing
- Test Case Design Techniques

Purpose:
This project demonstrates my knowledge of manual testing and test case writing for web applications.
